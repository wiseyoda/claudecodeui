# PR Assessments

This folder contains assessments of upstream PRs from [siteboon/claudecodeui](https://github.com/siteboon/claudecodeui).

## Assessment Template

For each PR, create a file named `PR-{number}-{short-title}.md`:

```markdown
# PR #{number}: {title}

**URL**: https://github.com/siteboon/claudecodeui/pull/{number}
**Author**: @{username}
**Status**: Open/Merged/Closed
**Assessment Date**: YYYY-MM-DD

## Summary
Brief description of what the PR does.

## Changes
- Key file changes
- New features/fixes

## Merge Recommendation
- [ ] Safe to merge
- [ ] Needs review
- [ ] Skip (conflicts with our changes)
- [ ] Wait for upstream to merge first

## Notes
Any additional context or concerns.
```
